Node.js unofficial client

eveai-api-unofficial

Gọi EverAI Text-To-Speech bằng cookie đăng nhập của chính bạn, dùng được như thư viện hoặc CLI.

npm install eveai-api-unofficial

Dùng như thư viện

Import package, truyền cookie hợp lệ, gọi textToSpeech và nhận audioUrl.

Không lưu secret

.env, cookie, key và file local đã được ignore để publish GitHub/npm an toàn hơn.

Có log đầy đủ

Bật LOG_LEVEL=info hoặc debug khi cần theo dõi request/polling.

Ví dụ nhanh

const { createClient } = require("eveai-api-unofficial");

const client = createClient({
  cookie: "csrftoken=...; sessionid=...",
  logLevel: "info",
});

const result = await client.textToSpeech({
  text: "Xin chào từ EverAI TTS",
  modelId: "everai-v1.5",
  audioType: "mp3",
});

console.log(result.audioUrl);

CLI

cp .env.example .env
npm run tts
node bin/eveai-tts.js --text "Xin chào" --format wav

Bảo mật

Cookie/session có quyền truy cập tài khoản EverAI, hãy xem như token đăng nhập. Không commit .env thật và không publish cookie lên npm.
